Hoe verloopt de bestelling?

  1. Klik op homepagina op “mysterie pick” product
  2. Er opent een pop-up scherm dat bevat:
    1. Size keuze
    2. Link naar club keuzes
    3. Mogelijkheid om change tokens bij te kopen
      1. Met een change token kan je de toegewezen truitjes wijzigen
  3. Na betaling kan je bij de bestellingsgeschiedenis (te vinden onder emailadres in navigatie):
    1. Naar je bestelling gaan
    2. Je shirts wijzigen door op het change token icoontje te duwen, er zijn 2 mogelijkheden:
      1. Je kan in 1 keer alle 3 de toegewezen truitjes wissen
      2. Of in de truitjes pop-up kan je 1 afzonderlijk truitje wijzigen
  4. Vanaf de volgende dag 0u kan je de truitjes niet meer veranderen!
  1. Klik op de homepagina op "Random shirt"
  2. Er opent een pop-up scherm dat bevat:
    1. Size keuze
    2. Link naar club keuzes
    3. Mogelijkheid om change tokens en cheat tokens bij te kopen
      1. Een cheat token is nodig hierbij als je het product al op voorhand wilt zien !!
      2. Met een change token kan je het truitje wijzigen
  3. Na betaling kan je bij de bestellingsgeschiedenis (te vinden onder emailadres in navigatie):
    1. Naar je bestelling gaan
    2. Je shirt zien door op het cheat token icoontje te duwen (Zorg dan wel dat je cheat tokens hebt)
    3. Je shirt wijzigen door op het change token icoontje te duwen
  4. Vanaf de volgende dag 0u kan je de truitjes niet meer veranderen!
